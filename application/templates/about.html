About

This is a Pokedex and Pokemon Team Maker.

Pokedex:

This is strictly for the sake of documenting Pokemon
and their characteristics. This include everything from
HP and Base Stat Totals to which generation each Pokemon
is from and how long it takes for their eggs to hatch.
It is desgined to make it easy for trainers to find
exactly what Pokemon they need for their teams.
If I decide to work on this project in the future
this will also include the newest generation Pokemon,
their moves, egg moves, level-up moves and much more.

Team Maker:

This part of the website is for users to implement
CRUD (create, read, update, delete) Pokemon for their
teams of 6 Pokemon. Ideally, I will implement a type
effectiveness chart so that it is easier for trainers
to analyse which types they need to work around.
